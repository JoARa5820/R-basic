변수에 2를 입력한 후 그 변수에 값이 2의 배수면 "2의 배수" 출력
아니면 "2의 배수가 아니다" 출력해주세요 (if문, ifelse함수)


[Answer]

x <- 2

if(x %% 2 == 0){
  print("2의 배수")
} else {
  print("2의 배수가 아니다")
}


ifelse(x %% 2 == 0, "2의 배수", "2의 배수가 아니다")	# 굳이 print 할 필요없음

> ifelse(x %% 2 == 0, "2의 배수", "2의 배수가 아니다")
[1] "2의 배수"
